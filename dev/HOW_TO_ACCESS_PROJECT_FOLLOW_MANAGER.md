# How to Access Project Follow Manager

## Quick Access Methods

### Method 1: From Project Detail Page (Recommended)
1. Navigate to any project detail page
2. Look for the **"Follow & Track"** button in the top toolbar (next to Export button)
3. Click the button to open the Project Follow Manager

**Visual Guide:**
```
[‚Üê Back] Project Title                    [Follow & Track] [Export] [Delete]
```

### Method 2: Direct URL
1. Navigate directly to: `/projects/{project_id}/follow`
   - Replace `{project_id}` with the actual project ID
   - Example: `/projects/123/follow`

### Method 3: From Dashboard
1. Go to your Dashboard
2. Click on any project card
3. On the project detail page, click **"Follow & Track"** button

## Quick Actions

### Follow/Unfollow a Project
- Click the **"Follow"** or **"Unfollow"** button in the top right
- Get notified about project updates
- Customize notification preferences

### Apply Milestone Template
- If no milestones exist, you can apply a template
- Templates define a sequence of milestones with estimated durations
- Automatically calculates due dates based on start date

### Create Custom Milestone
- Click **"Add Milestone"** button in Timeline tab
- Define title, description, due date, and dependencies
- Set milestone order

## Permissions

- **Project Creator**: Full access (create, edit, delete milestones)
- **Project Members**: Can view and update milestones
- **Advisors**: Can view, create flags, and resolve flags
- **Admins**: Full access to all features
- **Other Users**: View-only access

## Tips

1. **Use the Timeline** to get a quick overview of project progress
2. **Check Activity Feed** regularly to stay updated on project changes
3. **Monitor Health Score** to identify areas needing attention
4. **Create Flags** early when you notice potential issues
5. **Follow Projects** you're interested in to get notifications

## Troubleshooting

**Can't see the "Follow & Track" button?**
- Make sure you're logged in
- The button only appears for authenticated users
- Check if you have permission to view the project

**Can't edit milestones?**
- Only project creators, members, and admins can edit
- Check your role and project membership

**Timeline is empty?**
- No milestones have been created yet
- Apply a template or create milestones manually
- Contact project creator or admin to add milestones
